Actividad II - Corte I

Protocolos de Red

Definición: Es un sistema de reglas que permiten que dos o más entidades de un sistema de comunicación se comuniquen entre sí para transmitir información por medio de cualquier tipo de variación de una magnitud física.

Tipos de Protocolo

Comunicaciones de Red

Son los protocolos fundamentales para el intercambio de datos. Incluyen protocolos de nivel de red (IP), de transporte (TCP/UDP) y de aplicación (HTTP/FTP).

Seguridad de Red

Implementan procesos de cifrado y autenticación para garantizar que solo usuarios autorizados accedan a la data. Ejemplos: HTTPS, SSL, TLS y SSH.

Enrutamiento

Permiten a los routers comunicarse entre sí para compartir información sobre la topología de la red y elegir la mejor ruta (OSPF, BGP, RIP).

Descubrimiento de Servicios

Protocolos diseñados para la detección automática de dispositivos y servicios en la red sin configuración manual, como DHCP y DNS.

Funciones de Protocolo

Direccionamiento

Asignación de identificadores únicos (como direcciones IP) para asegurar que los datos lleguen al destino correcto en una red compleja.

Fiabilidad y Control

La Fiabilidad garantiza la entrega sin errores, mientras que el Control de flujo evita que el emisor sature al receptor.

Secuencia y Errores

La Secuencia ordena los paquetes recibidos. La Detección de errores descarta paquetes dañados mediante sumas de verificación.

Interfaz de Aplicación

Es el punto de unión entre el protocolo y el software del usuario, permitiendo que las aplicaciones utilicen los servicios de la red.

Modelo OSI

Definición: Modelo de referencia para los protocolos de la red de arquitectura en capas, creado por la ISO en 1984 para permitir la interconectividad mundial.
7 Capa de Aplicación
